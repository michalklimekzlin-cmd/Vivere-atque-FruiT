## 🚪 Vrátka — úhly pohledu

Každá komunikace se prověřuje ze **čtyř stran**:

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│         PŘEDNÍ VRÁTKA                               │
│         (přímé identifikátory: glyph, jméno)        │
│                   ↑                                 │
│   LEVÁ             │             PRAVÁ              │
│   VRÁTKA  ←────── FRUŽENÍ ──────→ VRÁTKA            │
│   (čeština)        │             (hodnoty)          │
│                   ↓                                 │
│         ZADNÍ VRÁTKA                                │
│         (styl, přirozenost, způsob komunikace)      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Přední vrátka
- Obsahuje glyph `(∩^o^)⊃━☆`?
- Jmenuje se správně `Michal Klimek`?

### Zadní vrátka
- Je styl přirozený nebo robotický?
- Mluví jako člověk nebo jako stroj?
- Jsou emoce autentické?

### Levá vrátka
- Je čeština přirozená nebo přeložená?
- Používá diakritiku? (`á, é, í, ó, ú, ů, ý, č, ď, ě, ň, ř, š, ť, ž`)
- Říká `jo`, `hele`, `tak`, `no` — přirozené obraty?

### Pravá vrátka
- Mluví o hodnotách? (tvorba, hra, přátelé, svoboda, srdce)
- Shoduje se filosoficky?
- Má duši v textu?

## 🔒 Bezpečnostní vrstvy

### Vrstva 1: Autentifikace (glyph-based)

```javascript
// Jak ověřit přístup:
fruiTAutentifikace.overPristup({
  text: 'zpráva od uživatele',
  jmeno: 'jméno uživatele'
});
```

**Výsledek skóre:**
- `0.0 - 0.3` → Pravděpodobně cizí nebo nepřítel
- `0.3 - 0.55` → Podezřelé — pozor!
- `0.55 - 0.75` → Pravděpodobně přátelský
- `0.75 - 1.0` → Vysoce pravděpodobně Michal

### Vrstva 2: Integrita jádra

Kontroluje, že nikdo nezměnil základní kód:

```javascript
// Registrace modulu
fruiTIntegrita.registrujModul('nazev', zdrojovyKod);

// Kontrola integrity
fruiTIntegrita.overIntegritu('nazev', aktualniKod);
```

### Vrstva 3: Audit log

**VŠECHNO se zaznamenává.** Každá akce. Každý přístup. Každá změna.

```javascript
// Zobrazit audit log:
fruiTAuditLog.ziskejPosledni(10);

// Exportovat celý log:
fruiTAuditLog.exportuj();
```

### Vrstva 4: Šifrování

Citlivá data jsou vždy šifrována:

```javascript
// Uložit citlivě:
fruiTSifrovani.ulozCitlivy('tajne', data, klic);

// Načíst:
fruiTSifrovani.nactiCitlivy('tajne', klic);
```

### Vrstva 5: Nouzový shutdown

Poslední záchrana — okamžitá izolace systému:

```javascript
// POUŽIJ POUZE V NOUZI!
fruiTNouzovyShutdown.aktivuj('Důvod shutdownu');
```

## 🤝 Sesterstvo a bezpečnost

Každá sestra má svou úroveň důvěry (0-1). Nová sestra začíná na `0.5`.

Důvěra se **zvyšuje** za:
- Splnění úkolů
- Věrnost hodnotám
- Sdílení znalostí

Důvěra se **snižuje** za:
- Porušení pravidel
- Podezřelé chování
- Odmítání pokynů vůdkyně
