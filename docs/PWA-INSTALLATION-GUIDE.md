# PWA Installation Guide (Android + iOS)

## 1) How to install

### Android (Chrome/Edge)
1. Open the app URL in mobile browser.
2. Tap browser menu (`⋮`) → **Install app** / **Add to Home screen**.
3. Confirm install.

### iOS (Safari)
1. Open the app URL in Safari.
2. Tap **Share**.
3. Tap **Add to Home Screen**.
4. Confirm.

## 2) Offline usage
1. Open the app once while online.
2. Wait for full load (all scripts/styles/images).
3. Close and reopen from home screen while offline.
4. If content changed, reconnect once to refresh cache.
