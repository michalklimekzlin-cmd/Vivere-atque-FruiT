# VaF'i'T Paměť — modulární verze

Nahraj obsah složky `docs` do složky `docs` v repozitáři.

Hlavní soubory:
- `index.html` — kostra aplikace
- `css/pamet.css` — vzhled
- `js/aplikace.js` — současná funkční logika
- `js/uloziste.js` — připravený modul úložiště
- `js/sloty.js` — připravený modul slotů
- `js/export-import.js` — připravený modul exportu
- `data/schema-pameti.json` — popis struktury Paměti

Současná funkce Paměti zůstává zachovaná. Pomocné moduly jsou připravené pro další bezpečné rozdělování logiky.
